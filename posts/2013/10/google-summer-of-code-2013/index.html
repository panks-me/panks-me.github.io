Google Summer of Code 2013

October 24, 2013

I mentored two projects for Google Summer of Code this year (2013), under KDE for digiKam project.

Check them out by fetching the latest kipi-plugin source from here: https://projects.kde.org/projects/extragear/graphics/kipi-plugins/repository :)

or linux users can also install digikam and kipi-plugin packages from your repository.

Cloud Integration Export Plugin

Description : Creating a kipi export plugin for putting the pictures on cloud storage devices like Google Drive, DropBox, etc.

Student : Saurabh Patel (saurabhpatel7717 at gmail dot com)

Proposal page : http://www.google-melange.com/gsoc/proposal/review/google/gsoc2013/saurabh_patel/1

Local Adjustment Tool in digiKam

Description : digiKam features an existing graphical interface tool “image editor” to make manipulations and user defined edits in an image. Apart from the local contrast tool, these edits apply to the entire image. The local adjustment tool would help users to make user defined selections in the image, based on the color of a point selected and a circle encircling the region identifying the region of interest of the user.

Student : Sayantan Datta (kenzo dot zombie at gmail dot com)

Proposal page : http://www.google-melange.com/gsoc/proposal/review/google/gsoc2013/sayantand/1
